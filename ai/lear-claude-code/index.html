🎯 学习路径 ​

入门阶段（s01-s04）：

  • 理解 Agent 循环的本质
  • 掌握工具调用机制
  • 学会任务追踪与子 Agent 委托

进阶阶段（s05-s08）：

  • 动态技能扩展
  • 上下文管理策略
  • 异步任务与后台执行

高级阶段（s09-s12）：

  • 多 Agent 协作设计
  • 团队协议与通信
  • 自主决策与任务隔离

💡 核心洞见 ​

"One loop & Bash is all you need"

Agent 开发的本质是：

  1. 循环：持续运行直到任务完成
  2. 工具：连接 LLM 与真实世界的桥梁
  3. 状态：追踪进度、管理上下文
  4. 协作：委托、并行、协调

📖 本系列基于 Claude Code 实战经验整理，适合有一定 Agent 开发基础的开发者深入学习。
