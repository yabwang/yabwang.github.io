## 💡 学习建议

### 面试准备要点

**必答知识点**：
1. Agent 架构组成（Memory、Planner、Tools）
2. RAG 原理与优化策略
3. Function Calling 执行流程
4. MCP 协议核心概念
5. Agent 评估维度与方法
6. Prompt Engineering 高级技巧

### 项目经验准备（STAR 法则）
- **Situation**：项目背景和挑战
- **Task**：你的任务和目标
- **Action**：具体的技术方案和实施
- **Result**：最终效果和数据指标

### 每周学习建议
- 完成当周知识点学习
- 完成实践任务并记录笔记
- 使用面试题库进行自测
- 准备 2-3 个项目案例用于面试

1. 阅读 [学习指南](/ai/agent-interview/learning-guide) 了解完整学习路线

3. 完成每周实践任务
