# ☕ Java 面试学习

> 系统性整理 Java 程序员面试核心知识点

## 📋 学习目标

本模块旨在帮助 Java 开发者系统性地准备技术面试，涵盖从基础到高级的完整知识体系。

## 📚 知识体系

### 核心基础
- **Java 基础**：语言特性、集合框架、IO/NIO
- **并发编程**：线程、JUC 包、并发设计模式
- **JVM**：[内存模型、GC 与类加载](/interview/jvm-basics)、性能调优

### 框架技术
- **Spring 生态**：IOC、AOP、Spring Boot、Spring Cloud
- **MyBatis**：工作原理、缓存机制、插件机制
- **其他框架**：Redis、消息队列、定时任务

### 数据库
- **MySQL**：索引原理、事务、锁机制、SQL 优化
- **Redis**：数据结构、持久化、缓存策略、分布式锁

### 分布式系统
- **分布式理论**：CAP、BASE、一致性协议
- **分布式组件**：服务注册、配置中心、网关、熔断降级
- **分布式事务**：Seata、TCC、Saga

### 系统设计
- **设计原则**：SOLID、设计模式
- **场景设计**：秒杀系统、限流、负载均衡、短链接

## 🎯 学习路径

1. **基础巩固**（第1-2周）
   - Java 基础、集合框架、并发编程基础

2. **深入理解**（第3-4周）
   - JVM、Spring 框架深入

3. **扩展学习**（第5-6周）
   - 数据库、Redis、分布式基础

4. **综合提升**（第7-8周）
   - 系统设计、项目经验总结

5. **面试准备**（第9-10周）
   - 模拟面试、查漏补缺

## 💡 学习建议

- **优先级排序**：根据目标岗位要求，优先学习核心技能
- **理论与实践结合**：不仅要理解原理，还要动手实践
- **项目经验梳理**：深入思考项目中遇到的问题和解决方案
- **技术深度**：选择1-2个方向深入研究，形成技术亮点
