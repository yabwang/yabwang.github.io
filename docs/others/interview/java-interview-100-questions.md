## 使用建议

### 快速复习法

1. **第一遍**：快速浏览所有题目，标记不熟悉的
2. **第二遍**：重点攻克标记题目，理解核心要点
3. **第三遍**：模拟面试，口头回答

### 深入学习法

1. 针对每个知识点，查阅官方文档或源码
2. 结合实际项目经验，准备案例
3. 整理成自己的面试笔记

### 记忆技巧

- **对比记忆**：如 ArrayList vs LinkedList
- **场景记忆**：结合使用场景理解
- **关键词记忆**：记住核心要点关键词
